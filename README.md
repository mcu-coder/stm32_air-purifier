
STM32空气净化器功能如下：
1.	MQ-2检测烟雾，超标声光报警，开启净化器
2.	PM2.5模块检测粉尘
3.	MQ-7检测一氧化碳燃气
4.	MQ-135检测空气质量
5.	MS1100检测甲醛值
6.	烟雾、燃气、空气质量、粉尘、甲醛超标触发蜂鸣器和指示灯报警并开启净化器
7.	DHT11检测环境温湿度，温度或者湿度超过阈值时语音播报对应提示词
8.	OLED实时显示当前环境数据
9.	按键可调节阈值、手动控制净化器开关
10.	通过蓝牙将数据发送到手机端，并可以控制风扇和净化器、模式切换
